# Expense Flow

Minimal personal expense dashboard built with Next.js 14 and Tailwind CSS. Add spending entries, monitor category distribution, and track six-month trends straight from the browser with local-only storage.

## Getting Started

```bash
npm install
npm run dev
```

Visit `http://localhost:3000` to use the dashboard.

## Scripts

- `npm run dev` – start the local development server
- `npm run build` – create a production build
- `npm run start` – run the production server locally
- `npm run lint` – lint the codebase with ESLint

## Deployment

The project is optimized for Vercel. Once you are ready, run:

```bash
npm run build
vercel deploy --prod --yes --token $VERCEL_TOKEN --name agentic-86658f0b
```
